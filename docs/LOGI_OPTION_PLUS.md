# Profil Logi Option + pour WebAmp

Ce document décrit la configuration du profil Logi Option + (MX Creative Console) pour WebAmp.

## Configuration des boutons

### Boutons 1-9 : Activation/Désactivation des pédales

Les boutons 1 à 9 de la MX Creative Console sont mappés aux raccourcis clavier `1` à `9` pour activer/désactiver les pédales dans l'ordre du pedalboard.

- **Bouton 1** → Touche `1` → Active/Désactive la première pédale
- **Bouton 2** → Touche `2` → Active/Désactive la deuxième pédale
- **Bouton 3** → Touche `3` → Active/Désactive la troisième pédale
- **Bouton 4** → Touche `4` → Active/Désactive la quatrième pédale
- **Bouton 5** → Touche `5` → Active/Désactive la cinquième pédale
- **Bouton 6** → Touche `6` → Active/Désactive la sixième pédale
- **Bouton 7** → Touche `7` → Active/Désactive la septième pédale
- **Bouton 8** → Touche `8` → Active/Désactive la huitième pédale
- **Bouton 9** → Touche `9` → Active/Désactive la neuvième pédale

### Boutons supplémentaires (optionnels)

Vous pouvez configurer d'autres boutons de la MX Creative Console pour les actions suivantes :

- **Ouvrir la bibliothèque** : `Ctrl+L`
- **Ouvrir les paramètres** : `Ctrl+,`
- **Ouvrir les profils** : `Ctrl+P`
- **Sauvegarder le profil** : `Ctrl+S`
- **Vider le pedalboard** : `Ctrl+Shift+C`
- **Métronome** : `Ctrl+M`
- **Looper** : `Ctrl+Shift+L`
- **Tuner** : `Ctrl+T`
- **Machine à rythmes** : `Ctrl+D`
- **Simulateur de pièce** : `Ctrl+R`

## Configuration dans Logi Option +

1. Ouvrez **Logi Option +**
2. Sélectionnez votre **MX Creative Console**
3. Créez un nouveau profil nommé **"WebAmp"**
4. Pour chaque bouton (1-9), configurez :
   - **Type d'action** : Touche de clavier
   - **Touche** : `1`, `2`, `3`, etc. (selon le bouton)
   - **Modificateurs** : Aucun

5. Pour les boutons supplémentaires (si disponibles), configurez les raccourcis correspondants avec les modificateurs appropriés (Ctrl, Shift, etc.)

## Utilisation

Une fois le profil configuré dans Logi Option + :

1. Ouvrez WebAmp dans votre navigateur
2. Activez le profil "WebAmp" dans Logi Option +
3. Les boutons 1-9 activeront/désactiveront automatiquement les pédales correspondantes dans l'ordre du pedalboard

## Notes

- Les raccourcis peuvent être personnalisés dans WebAmp via **Paramètres → Configurer les raccourcis**
- Le mode "clavier uniquement" peut être activé pour une navigation complète sans souris
- Les raccourcis fonctionnent même si vous êtes dans un champ de saisie (pour les raccourcis avec modificateurs)

## Compatibilité

Ce profil est compatible avec :
- **Logitech MX Master 3S** (avec MX Creative Console)
- **Logitech MX Keys Mini** (si configuré avec des raccourcis)
- Toute souris/clavier Logitech compatible avec Logi Option +
